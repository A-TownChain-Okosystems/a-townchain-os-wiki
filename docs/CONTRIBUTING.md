# Contributing Guide

Vollständige Anleitung: [CONTRIBUTING.md im Haupt-Repo](https://github.com/A-TownChain-Okosystems/a-townchain-os/blob/main/docs/CONTRIBUTING.md)

## Schnellstart
```bash
git clone https://github.com/A-TownChain-Okosystems/a-townchain-os
cd a-townchain-os
pip install -r requirements.txt
python -m pytest tests/  # Alle Tests grün?
```

## Wiki-Beiträge
- Neue Wiki-Seite: in entsprechendem `*-wiki` Repo anlegen
- Format: Markdown, deutsch oder englisch
- Codebeispiele: ATCLang wo möglich

## Commit-Stil
`feat:` / `fix:` / `sec:` / `docs:` / `perf:` / `test:` + kurze Beschreibung
